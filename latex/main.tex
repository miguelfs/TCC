  \begin{abstract}

  Apresenta-se, neste trabalho de conclusão de curso, o comparativo entre
 Sistemas de Recomendação Baseados em Sessão (SBRS)  para recomendar convites de
 participação de usuários no aplicativo Indaband, com as sessões de gravação como objeto de
 estudo, abordando o funcionamento de modelos como regras de associação, kNNs e
 arquiteturas de redes neurais, além da apresentação dos resultados da avaliação
 objetiva baseada no histórico de dados disponível.
  \end{abstract}

  \begin{foreignabstract}

  In this work, we present a comparison between Session-Based Recommendation
  Systems (SBRS) for recommending user invitations for musical sessions within
  Indaband mobile application. It addresses the functioning of models such as
  association rules, kNNs, and neural network architectures, along with
  evaluating objective metrics over the available data history.

  \end{foreignabstract}
